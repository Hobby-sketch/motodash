📁 Setup Project Lokal
~5 menit

Langkah-Langkah:

① Extract file ZIP

Extract motodash.zip ke folder pilihan Anda. Hasilnya:

motodash/
├── index.html
├── manifest.json
├── sw.js
├── css/style.css
├── js/(8 file JS)
├── assets/icons/
│   ├── icon.svg
│   ├── icon-192.png  ← dari Step 2
│   └── icon-512.png  ← dari Step 2
├── .github/workflows/deploy.yml
├── .nojekyll
└── README.md

② Tes di lokal dulu (opsional tapi disarankan)

cd motodash
python3 -m http.server 8080

Buka browser: http://localhost:8080

💡 Harus pakai local server (bukan buka file HTML langsung) agar GPS & PWA features bekerja.

③ Init Git di folder project

cd motodash
git init
git add .
git commit -m "feat: MotoDash v1.0.0 initial commit"
  • File ZIP sudah di-extract
  • Icon PNG sudah ditaruh di assets/icons/
  • Git init sudah dijalankan

② Push Code ke GitHub

Copy dan jalankan perintah ini di terminal (ganti username jika belum diisi di atas):

git remote add origin https://github.com/USERNAME/motodash.git
git branch -M main
git push -u origin main
🔐 GitHub mungkin meminta login. Gunakan Personal Access Token sebagai password. Buat di: Settings → Developer settings → Tokens

③ Verifikasi Push Berhasil

5
🌐 Aktifkan GitHub Pages
~2 menit

Metode A — GitHub Actions (Otomatis, Disarankan)

Repository sudah berisi file .github/workflows/deploy.yml. Aktifkan dengan langkah berikut:

  1. Buka repository → tab Settings
  2. Sidebar kiri → klik Pages
  3. Di bagian "Build and deployment" → Source: pilih "GitHub Actions"
  4. Klik Save
  5. Kembali ke tab Actions → tunggu workflow selesai (✅ hijau)
  6. URL live akan muncul di tab Pages
✅ Setelah ini, setiap kali Anda push ke main, app akan ter-deploy otomatis!

URL Aplikasi Anda:

https://USERNAME.github.io/motodash/

6
📱 Install PWA di Android
Langkah Terakhir
📱 Gunakan Chrome Android untuk hasil terbaik. Buka URL dari Step 5 di Chrome Android.

Cara Install (Chrome Android)

  1. Buka Chrome Android → navigasi ke URL aplikasi
  2. Tunggu sampai aplikasi loading penuh (peta muncul)
  3. Tap ikon ⋮ (3 titik) di pojok kanan atas Chrome
  4. Tap "Add to Home screen" atau "Install app"
  5. Konfirmasi nama MotoDash → tap Add
  6. Icon MotoDash muncul di home screen 🎉
✅ Setelah install, aplikasi akan berjalan fullscreen landscape tanpa browser bar — persis seperti native app!

Tips Penggunaan di Motor

  • Mount HP di handlebar mount landscape mode
  • Aktifkan Brightness 100% di Settings panel
  • Aktifkan GPS High Accuracy sebelum berangkat
  • Pasang headset via Bluetooth panel untuk navigasi suara
  • Set destination via Search bar atau perintah suara
  • Aplikasi berhasil diinstall di home screen
  • GPS speedometer menampilkan kecepatan
  • Peta maps terbuka dan bisa di-zoom
  • Navigasi ke suatu tempat berhasil
