MOTODASH

Tutorial Singkat — Dari Download Sampai Jalan di Motor

1
Putus Koneksi Git Lama
Jika sudah pernah push
ℹ️

Jika belum pernah push ke GitHub sebelumnya, lewati step ini dan langsung ke Step 2.

A
Cek apakah ada koneksi git lama
git remote -v

Jika muncul URL github.com → ada koneksi lama yang harus diputus.

B
Pilih salah satu cara memutus:

✅ Cara mudah

Pakai folder baru dari ZIP yang baru di-extract. Tidak perlu lakukan apapun — langsung ke Step 2.

Hapus .git manual

Jika ingin pakai folder yang sama, hapus folder .git:

# Mac / Linux: rm -rf .git # Windows PowerShell: Remove-Item -Recurse -Force .git
C
Hapus repo lama di GitHub (opsional)

GitHub → repo lama → Settings → scroll bawah → Danger Zone → Delete this repository

2
Extract File & Cek Lokal
~5 menit
1
Extract motodash_final.zip

Simpan di lokasi mudah, contoh: C:\Projects\motodash\ atau ~/Projects/motodash/

2
Buka Terminal → masuk ke folder
# Mac/Linux: cd ~/Projects/motodash # Windows: cd C:\Projects\motodash
3
Jalankan local server
python3 -m http.server 8080

Windows tanpa Python: npx serve . -p 8080

4
Buka Chrome → ketik di address bar:
http://localhost:8080

Dashboard harus muncul dengan peta OpenStreetMap di panel kanan. ✅

🚫

Jangan buka file HTML langsung (double-click). GPS dan fitur lain tidak akan bekerja tanpa server HTTP.

3
Generate Icon PNG
Wajib untuk Install
⚠️

Tanpa icon PNG 192×192 dan 512×512, Chrome Android tidak akan menampilkan tombol "Add to Home Screen".

1
Buka halaman generator (server harus jalan)
http://localhost:8080/generate-icons.html
2
Klik "Generate Icons" → Download 2 file:

⬇ icon-192.png  ⬇ icon-512.png

3
Simpan kedua file ke:
motodash/assets/icons/icon-192.png motodash/assets/icons/icon-512.png
4
Push ke GitHub
~10 menit
1
Set identitas Git (sekali saja per komputer)
git config --global user.name "Nama Anda" git config --global user.email "email@anda.com"
2
Buat repository baru di GitHub

Buka github.com/new

Repository namemotodash
Visibility✅ Public (wajib untuk GitHub Pages gratis)
Initialize README❌ Jangan dicentang

→ Klik Create repository

3
Jalankan perintah ini di terminal (ganti USERNAME)
git init git add . git commit -m "feat: MotoDash v1.0.0" git remote add origin https://github.com/USERNAME/motodash.git git branch -M main git push -u origin main
🔑

Saat diminta password → paste Personal Access Token, bukan password GitHub biasa.
Buat token di: github.com/settings/tokens/new → centang repo → Generate.

4
Verifikasi — buka browser
https://github.com/USERNAME/motodash

Semua file harus terlihat di sana. ✅

5
Aktifkan GitHub Pages
~2 menit
1
Buka repository → klik tab Settings
2
Sidebar kiri → Pages
3
Build and deployment → Source → pilih "GitHub Actions"
4
Klik tab "Actions" → tunggu workflow selesai ✅

(biasanya 1–2 menit, muncul centang hijau)

5
URL aplikasi Anda sudah live:
https://USERNAME.github.io/motodash/
🎉

Aplikasi online! Setiap kali push, auto-deploy dalam 1–2 menit.

⚠
Jika Actions gagal (❌ merah)

Settings → Actions → General → Workflow permissions → Read and write → Save → push ulang:

git commit --allow-empty -m "retry deploy" git push
6
Install di Android
Langkah Terakhir
1
Buka Chrome Android
https://USERNAME.github.io/motodash/
2
Tunggu peta muncul (loading penuh dulu)
3
Tap menu ⋮ → "Add to Home Screen" atau "Install app"
4
Tap Add → icon muncul di home screen 🎉
5
Izinkan semua akses saat diminta:
📍 LokasiAllow while using app
🎤 MikrofonAllow
📡 BluetoothAllow
⚠️

Tombol install tidak muncul? Pastikan icon-192.png dan icon-512.png sudah ada di assets/icons/ dan sudah di-push ke GitHub.

— TROUBLESHOOTING —
🔧
Masalah Umum & Solusi
MasalahSolusi
Peta tidak munculCek koneksi internet → refresh → atau Settings → Clear Cache
GPS tetap SEARCHINGKeluar ruangan, langit terbuka. GPS butuh sinyal satelit.
Izin lokasi ditolakHP Settings → Apps → Chrome → Permissions → Location → Allow
Voice tidak jalanHarus Chrome/Edge. Izinkan akses mikrofon saat diminta.
BT tidak connectHarus HTTPS + Chrome. Masukkan headset ke mode pairing.
PWA tidak bisa installPastikan icon-192.png & icon-512.png ada di assets/icons/ dan sudah di-push.
404 setelah deployTunggu 2–5 menit. Cek Actions sudah ✅. Cek Pages sudah aktif.
Push rejectedGunakan Personal Access Token, bukan password GitHub biasa.
🔍

Untuk debug: tekan F12 di Chrome → tab Console. Tidak ada error merah = aplikasi normal.
