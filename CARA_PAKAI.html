Fase A — Persiapan di Komputer
A1

Extract File MotoDash

1
Download motodash_final.zip dari Claude

Simpan di tempat mudah, contoh: C:\ atau Desktop

2
Klik kanan file ZIP → "Extract All..."

Pilih lokasi extract: disarankan C:\motodash\
(hindari path dengan spasi, contoh jangan di "My Documents")

3
Pastikan isi folder sudah benar
motodash\ ├── index.html ← aplikasi utama ├── manifest.json ├── sw.js ├── generate-icons.html ← ⬅ akan dipakai di step A2 ├── CARA_PAKAI.html ← panduan ini ├── css\ js\ assets\ vendor\ ← WAJIB ikut diupload (Leaflet, font, dll) └── .github\workflows\deploy.yml
A2

Generate Icon PNG — Cukup Double-Click!

✨ VERSI TERBARU — Tidak perlu Python, tidak perlu server

File generate-icons.html sudah bisa langsung dibuka dengan double-click. Icon dibuat langsung di browser, tidak perlu koneksi internet apapun.

1
Masuk ke folder motodash di File Explorer

Cari dan buka folder C:\motodash\ (atau lokasi extract tadi)

2
Double-click file generate-icons.html

Browser Chrome/Edge akan terbuka otomatis menampilkan halaman generator

3
Klik tombol "🎨 Generate Icon Sekarang"

Dua preview icon MotoDash akan muncul dalam 1–2 detik

4
Klik download kedua file:

⬇ icon-192.png
⬇ icon-512.png

5
Pindahkan kedua file PNG ke folder yang benar:
C:\motodash\assets\icons\

Setelah dipindah, folder assets\icons\ berisi 3 file:

icon.svg ← sudah ada dari awal icon-192.png ← baru dipindahkan ✅ icon-512.png ← baru dipindahkan ✅
⚠️

Jika browser memblokir download (muncul popup "blocked") → klik kanan gambar icon → Save image as → simpan manual ke folder assets\icons\

A3

Install Git (jika belum ada)

1
Cek apakah Git sudah terinstall

Tekan tombol Windows di keyboard → ketik cmd → Enter → Command Prompt terbuka. Ketik:

git --version

Jika muncul git version 2.x.x → Git sudah ada, lewati ke Fase B.

2
Jika Git belum ada, download dari:

git-scm.com/download/win → download installer → jalankan → klik Next terus sampai selesai (semua pengaturan biarkan default)

3
Setelah install, tutup dan buka ulang Command Prompt

Verifikasi lagi dengan git --version → harus muncul versinya ✅

ℹ️

Semua perintah Git di tutorial ini dijalankan di Command Prompt (cmd) — bukan Git Bash, bukan PowerShell. Keduanya bisa, tapi cmd paling sederhana untuk pemula.

Fase B — GitHub

Buat Akun & Repository GitHub

Buka browser → pergi ke github.com/signup

Buat repository baru

Login GitHub → klik tombol + di pojok kanan atas → New repository

Isi form seperti ini:
Repository namemotodash

Add .gitignore❌ Jangan dicentang

→ Klik tombol hijau Create repository

B2

Buat Personal Access Token

ℹ️

GitHub tidak menerima password biasa untuk upload kode lewat terminal. Anda perlu membuat "token" khusus sebagai pengganti password.

1
Buka halaman token GitHub:

Di GitHub → klik foto profil (kanan atas) → Settings
Scroll sidebar kiri ke paling bawah → Developer settings
→ Personal access tokens → Tokens (classic)
→ Generate new token (classic)

2
Isi form token:
NoteKetik: motodash-deploy
ExpirationPilih: No expiration
Select scopes✅ Centang repo (paling atas)

→ Scroll ke bawah → klik Generate token

3
COPY TOKEN SEKARANG — hanya tampil sekali!

Token akan muncul berwarna hijau/kuning. Klik ikon copy di sebelahnya. Simpan di Notepad. Jika halaman ditutup, token tidak bisa dilihat lagi dan harus buat ulang.

B3

Setup Identitas Git (Sekali Saja)

1
Buka Command Prompt baru

Tekan Windows → ketik cmd → Enter

2
Ketik perintah ini — ganti dengan nama Anda:
git config --global user.name "Nama Anda"
3
Ketik perintah ini — ganti dengan email GitHub Anda:
git config --global user.email "email@anda.com"
B4

Upload Kode ke GitHub

1
Buka Command Prompt DI DALAM folder motodash

Cara paling mudah:

1. Buka File Explorer
2. Masuk ke folder C:\motodash\
3. Klik kotak alamat (address bar) di atas — kotak yang biasanya menampilkan path
4. Hapus teksnya, ketik cmd, tekan Enter
5. Command Prompt terbuka langsung di dalam folder motodash ✅

2
Pastikan sudah di folder yang benar
cd

Hasilnya harus menampilkan path yang berakhir dengan \motodash

3
Jalankan 6 perintah ini — satu per satu, tekan Enter tiap selesai:

Perintah 1 — buat folder ini jadi git repository:

git init

Perintah 2 — tambahkan semua file:

git add .

Perintah 3 — simpan perubahan:

git commit -m "MotoDash v1.0.0"

Perintah 4 — sambungkan ke GitHub (GANTI USERNAME dengan username GitHub Anda):

git remote add origin https://github.com/USERNAME/motodash.git

Perintah 5 — set nama branch:

git branch -M main

Perintah 6 — upload ke GitHub:

git push -u origin main
4
Saat diminta login setelah perintah ke-6:
Username:Ketik username GitHub Anda (bukan email)
Password:Paste Personal Access Token dari Step B2
💡

Saat paste token, teks tidak terlihat di terminal — itu normal dan aman. Paste saja lalu tekan Enter.

5
Verifikasi upload berhasil

Buka Chrome → pergi ke: https://github.com/USERNAME/motodash
Semua file harus terlihat di sana (index.html, css/, js/, dll) ✅

Fase C — Deploy & Install
C1

Aktifkan GitHub Pages (Hosting Gratis)

1
Buka repository di GitHub

Pergi ke: https://github.com/USERNAME/motodash

2
Klik tab "Settings" (ada ikon ⚙)

Letaknya di menu atas repo, paling kanan

3
Di sidebar kiri → scroll → klik "Pages"
4
Bagian "Build and deployment" → "Source"

Ubah dari "Deploy from a branch" → pilih "GitHub Actions"

5
Klik tab "Actions" di menu atas repo

Terlihat workflow sedang berjalan. Tunggu hingga muncul ✅ tanda centang hijau (1–3 menit)

6
Kembali ke Settings → Pages

URL aplikasi Anda sudah muncul:

https://USERNAME.github.io/motodash/
🎉

Aplikasi sudah online! Buka URL di atas untuk memastikan.

⚠
Jika Actions gagal (tanda ❌ merah):

Settings → Actions → General → Workflow permissions → pilih "Read and write permissions" → Save

Lalu jalankan di Command Prompt:

git commit --allow-empty -m "retry deploy" git push
C2

Install di HP Android

1
Buka Chrome di HP Android

Ketik URL ini di address bar:

https://USERNAME.github.io/motodash/
2
Tunggu peta gelap muncul di panel kanan

Jangan install sebelum peta OpenStreetMap muncul

3
Tap ⋮ (tiga titik) di pojok kanan atas Chrome
4
Tap "Add to Home screen" atau "Install app"

Teksnya beda-beda tergantung versi Chrome. Bisa juga muncul banner di bawah layar secara otomatis.

5
Tap "Add" → icon MotoDash muncul di home screen

Buka icon itu → aplikasi jalan fullscreen landscape tanpa browser bar! ✅

6
Izinkan semua akses saat diminta:
📍 Lokasi / LocationPilih "Allow while using app"
🎤 Mikrofon / MicrophonePilih "Allow"
📡 Bluetooth / Nearby devicesPilih "Allow"
⚠️

Tombol install tidak muncul? Pastikan icon-192.png dan icon-512.png sudah ada di folder assets\icons\ dan sudah di-upload ke GitHub (Step A2 + B4).

Fase E — Jika Ada Masalah
🔧

Solusi untuk Masalah Umum

MasalahSolusi
fatal: not a git repository Belum masuk folder motodash. Buka cmd DI DALAM folder motodash (cara: File Explorer → folder motodash → klik address bar → ketik cmd → Enter), lalu jalankan git init dulu.
Peta tidak muncul Cek koneksi internet. Refresh halaman. Atau Settings (di app) → Clear Cache → refresh.
GPS tetap SEARCHING Pergi ke luar ruangan. GPS HP butuh langit terbuka untuk menangkap sinyal satelit.
Izin lokasi ditolak HP Settings → Apps → Chrome → Permissions → Location → Allow.
git push meminta password Masukkan Personal Access Token dari Step B2 — bukan password GitHub biasa.
GitHub Actions ❌ merah Settings repo → Actions → General → "Read and write permissions" → Save. Lalu git commit --allow-empty -m "retry" dan git push.
404 setelah deploy Tunggu 3–5 menit. Cek tab Actions → workflow harus sudah ✅ hijau.
Tombol install tidak muncul di Chrome Android Pastikan icon-192.png dan icon-512.png ada di assets\icons\ dan sudah ter-upload ke GitHub.
generate-icons.html tidak mau download Klik kanan gambar icon yang muncul → Save image as → simpan manual ke assets\icons\.
Voice tidak merespons Harus Chrome/Edge. Pastikan izin mikrofon sudah Allow.
🔍

Cara cek error di browser: tekan F12 → klik tab Console. Jika ada tulisan merah, screenshot dan kirim ke Claude untuk bantuan lebih lanjut.
